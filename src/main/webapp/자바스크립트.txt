주제1. 자바스크립트(JavaScript)란?

1. **정의**:  
   - JavaScript는 웹 개발을 위해 가장 널리 사용되는 **스크립트 언어**입니다.
   - 웹 브라우저에서 실행되는 언어로, 사용자(클라이언트)와 웹페이지가 상호작용할 수 있도록 돕습니다.

2. **주요 역할**:
   - HTML: 웹페이지의 구조를 만듭니다.
   - CSS: 웹페이지의 디자인(색상, 글꼴, 레이아웃)을 설정합니다.
   - JavaScript: 웹페이지에 **동작**과 **기능**을 추가합니다.

3. **할 수 있는 일**:
   - 버튼을 클릭했을 때 특정 기능 실행.
   - 사용자 입력(예: 이름, 이메일)을 받아 처리.
   - 웹페이지의 내용(텍스트, 이미지 등)을 실시간으로 변경.
   - 애니메이션 효과 추가.

4. **브라우저에서 동작**:
   - JavaScript는 웹 브라우저에 내장된 **JavaScript 엔진**(예: Chrome의 V8)을 통해 실행됩니다.  
   - 별도의 설치가 필요하지 않고, 모든 최신 웹 브라우저에서 바로 사용할 수 있습니다.


### 자바스크립트(JavaScript)란?  비유를 통한 이해:
  - **HTML**이 뼈대(골격),
  - **CSS**가 옷(외형)이라면,  
  - **JavaScript**는 그 웹페이지를 움직이고 말하고 반응하게 만드는 마법 같은 도구(언어)입니다.


5.  자바스크립트(JavaScript)문법 중에서 ~   <script></script> 태그란 무엇인가요?

      <script></script>는 HTML 문서 안에 JavaScript 코드를 작성하거나 
                          외부 JavaScript 파일을 연결할 때 사용하는 태그입니다.


6. 왜 <script> 태그가 필요한가요?

    웹 페이지는 기본적으로 HTML과 CSS만으로는 구조(골격)와 디자인(옷)을 만들 수 있어요. 
    하지만, JavaScript를 사용하면 웹 페이지에 동작과 기능을 추가할 수 있습니다.
    이 JavaScript 코드를 HTML 문서 안에서 작성하거나 연결하려면 <script> 태그가 필요합니다.

  역할1. JavaScript 코드 삽입
          <script> 태그 안에 JavaScript 코드를 작성하면, 그 코드는 웹 브라우저에 의해 실행됩니다.
  역할2. 외부 파일 연결
            외부에 저장된 JavaScript 파일(.js 파일)을 HTML 문서에 연결해 사용할 수 있습니다.



6.1. 어떻게 사용하나요?

        사용방법1. HTML 안에 JavaScript 작성

              아래는 HTML 문서 안에 직접 JavaScript 코드를 작성하는 예입니다:

              <!DOCTYPE html>
              <html>
              <head>
                  <title>JavaScript 예제</title>
              </head>
              <body>
                  <h1>안녕하세요!</h1>
                  <script>
                      console.log("JavaScript 실행 중!");
                  </script>
              </body>
              </html>

  설명:
      <script> 태그 안에 작성된 console.log("JavaScript 실행 중!");는 개발자 도구의 콘솔에 메시지를 출력합니다.
--------------------------------------------------------------------------------------------------------

        사용방법2. 외부 JavaScript 파일 연결

                  JavaScript 코드를 HTML 파일과 분리하여 별도의 .js 파일로 저장할 수도 있습니다.

                  예:

                      HTML 파일 코드 내용:

                                <!DOCTYPE html>
                                <html>
                                <head>
                                    <title>외부 JavaScript 연결</title>
                                </head>
                                <body>
                                    <h1>안녕하세요!</h1>
                                    <script src="script.js"></script>
                                </body>
                                </html>



                      JavaScript 파일(script.js) 코드 내용:

                                console.log("외부 JavaScript 파일 실행 중!");

        설명:

            HTML 파일에서 <script src="script.js"></script>로 외부 파일을 불러옵니다.
            브라우저는 script.js 파일을 실행합니다.

주제3. 자바스크립트 용어와 기본 입출력 방법


설명 : 자바스크립트 언어를 본격적으로 공부하기 전에 프로그래밍과 관련된 몇 가지 기본 용어를 알아보자.
       그리고 자바스크립트 문법을 배우기 전에 프로그램을 실행하는 데 필요한 자료(데이터)를 입력하고
       입력한 자료(데이터)를 출력해서 확인 하는 방법도 알아보자.


  1. 자바스크립트에서 "식"(Expression)이란?

      **"식"(Expression)**은 값을 만들어내는 코드 조각입니다.
      자바스크립트에서 식은 특정 작업을 수행하고 결과로 하나의 값을 반환합니다.
      
        "식" 예1 )  3 + 5

                설명 : 여기서 3 + 5는 식입니다.
                      계산된 결과로 값 8이 반환됩니다.


        "식" 예2 ) "안녕하세요?"
          
                설명 : 안녕하세요?"는 문자열 리터럴로, 그 자체로 문자열 값을 나타내기 떄문에 식입니다.


   2. 자바스크립트의 문(Statement)이란?

      - 문(Statement)은 자바스크립트에서 특정 작업을 지시하는 코드 단위입니다.
        쉽게 말해, **"명령문"**으로 생각할 수 있습니다.

      - 문(Statement): 특정 작업을 실행하며, 보통 하나 이상의 식을 포함합니다.

      - 문의 끝에는 문의 끝을 나타내는 기호 세미콜론 ; 을 붙여서 구분해서 적습니다.

        "문" 예1 )  let x = 5;
     
                
        "문" 예2 )   if (true) console.log("Hello");

      3.5. 웹브라우저 개발자도구 화면(console탭)창에 출력을 담당하는 console.log() 명령문

            1. console.log()는 무엇인가요?

            console.log()는 컴퓨터에게 "이 내용을 화면에 보여줘"라고 말하는 명령이에요. 
            하지만 이 화면은 우리가 평소에 보는 웹사이트의 화면이 아니라, 
            개발자가 문제를 해결하거나 코드가 잘 작동하는지 확인할 수 있도록 만든 "개발자 도구"라는 특별한 화면이에요.

            2. 왜 사용하는 건가요?
            만약 요리를 한다고 생각해보세요. 
            요리가 다 됐는지 확인하려면 중간중간 맛을 보거나 재료가 제대로 들어갔는지 확인해야겠죠? 
            console.log()는 그런 확인 과정이에요. 
            코드가 잘 작동하고 있는지, 어떤 값이 들어가 있는지 확인하려고 사용해요.

            3. 어디서 확인할 수 있나요?
                웹 브라우저(예: 크롬, 엣지)에서 웹사이트를 열어요.
                오른쪽 클릭 > 검사를 선택하거나 F12 키를 눌러서 개발자 도구를 열어요.
                개발자 도구에서 "콘솔(Console)"이라는 탭을 클릭하면 console.log()로 출력된 내용이 보여요.
